Hive-JSON-Serde by rcongiu

Read/Write JSON SerDe for Apache Hive

SerDe to read/write JSON data. Data inserted into a table configured to use this serde will be automatically converted to JSON. Supports arbitrarily nested JSON structures.

Install

Copy json-serde-jar-with-dependencies-1.0.jar possibly to your $HIVE_EXTRA_LIBS directory, or wherever you want, and add the jar in the query with add jar '/path/to/serde';
